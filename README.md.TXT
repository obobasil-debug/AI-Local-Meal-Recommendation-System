# AI-Based Local Meal Recommendation System

This project is an AI-powered local meal recommendation system developed using Python and Streamlit.

## Project Description
The system recommends suitable local meals based on user preferences using a trained machine learning model.  
The dataset used for training was synthetically generated.

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy

## How to Run the Application
1. Install Python 3.8+
2. Install required dependencies
3. Run the app using:
   streamlit run app.py
